# Reald Tools

Petit pack d'outils Linux créé par Reald.

## Commandes
- `clean-system` : nettoie le système (apt, journaux, /tmp)
- `sys-health` : affiche un résumé santé du système
- `encrypt-dir` : chiffrer/déchiffrer un dossier
- `install.sh` : installe les commandes dans `/usr/local/bin`

## Installation (depuis GitHub)

curl -sSL https://raw.githubusercontent.com/TON_USERNAME/reald-tools/main/install.sh | bash
